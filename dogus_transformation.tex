\section{In-Scope Use Cases}

Three workstreams are in scope. This is the actual, agreed-with-customer scope, not a generic ten-agent menu.

\subsection{Workstream A, Lead Qualification (paid Phase 1)}

Outbound, VW Binek brand only at launch. Freya calls leads from a 450K to 500K customer dial pool, with about 150K-160K reached in a typical year. Driver: summer (Jul-Aug) +30 to 40\% volume spike that today saturates a 12-person peak shift.

\textbf{Conversation logic, locked April 15.} Confirm form data → confirm or change model preference → assign nearest VW Binek dealer using the JSON file (city priority, with Anadolu / Avrupa side split for Istanbul) → set appointment.
